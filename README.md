# HEVD-Exp

My personal writeups and exploit code for the [HackSys Extreme Vulnerable Driver (HEVD)](https://github.com/hacksysteam/HackSysExtremeVulnerableDriver).

## About

This repository contains hands-on solutions I wrote while working through HEVD challenges on modern Windows.  
Each folder targets a specific vulnerability type and includes the exploit source, relevant notes, and sometimes a brief walkthrough.

## Environment

- Windows 7 x86 Sp1 
- Windows 10 / 11 (x64)
- Visual Studio 2019
- Local kernel debugging (WinDbg)
